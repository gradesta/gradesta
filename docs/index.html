Gradesta

Stitchable spreadsheets for the 21st century

Gradesta is a protocol and ecosystem for creating dynamic graph based stitchable spreadsheet applications. This protocol has a wide array of applications from VR, to traditional screen based computing, to audio native or braille native computing. Because the protocol serves structure rather than an exact image, applications which serve this protocol can be hooked up to work well with almost any form factor.

Since this protocol gives access to data structures directly, it can also be used as a very low cost user interface for debugging purposes (giving you the ability to rapidly gain access to the internals of a running application) as well as develop very low cost user interfaces for bespoke applications such as business internal systems (CRM, custom inventory management, ect.)
